This is the source code for the Naturalis OAI-PMH framework. The
framework can access and manage multiple OAI repositories exposes their
data through a REST API that produces OAI-PMH XML.

Currently, the only OAI repository exposed through this used to
transfer DNA extract data from Geneious to CRS.

For the javadocs, see here:
http://naturalis.github.io/nl.naturalis.oaipmh/javadoc/
